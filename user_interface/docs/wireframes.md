## Key UI Patterns

### Card Layout Pattern
- All dashboard components use shadcn/ui Card system
- Consistent spacing with gap-4 grid layouts
- Responsive breakpoints using container queries (@container/main)

### Real-time Updates Pattern
- Components fetch initial data via API calls
- WebSocket connections provide live updates
- Loading states with skeleton components
- Error boundaries with retry options

### Navigation Pattern
- Sidebar-based navigation using shadcn/ui Sidebar
- Collapsible on mobile (offcanvas mode)
- Context-aware active states

### Form Interaction Pattern
- Drawer components for detailed views
- Modal dialogs for critical actions
- Consistent field layouts using shadcn/ui Field components
