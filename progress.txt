# CONFIRMED WORKING APIs - TURKEY + UK FOCUS

## 1. USGS Earthquake API ⭐ FREE, NO KEY
Turkey: https://earthquake.usgs.gov/fdsnws/event/1/query?format=geojson&limit=10&minlatitude=36&maxlatitude=42&minlongitude=26&maxlongitude=45
UK: https://earthquake.usgs.gov/fdsnws/event/1/query?format=geojson&limit=10&minlatitude=50&maxlatitude=60&minlongitude=-8&maxlongitude=2
Fields: mag, place, time, coordinates [lon,lat,depth], tsunami flag

## 2. EMSC Seismic Portal ⭐ FREE, NO KEY
Endpoint: https://www.seismicportal.eu/fdsnws/event/1/query
Example: ?limit=10&format=json&minlat=36&maxlat=42&minlon=26&maxlon=45
Best for: Turkey + UK/Europe earthquakes

## 3. AFAD Turkey Earthquakes ⭐ FREE, NO KEY
Endpoint: https://deprem.afad.gov.tr/EventData/GetEventsByFilter
Method: POST with JSON body
Fields: magnitude, coordinates, depth, location in Turkish

## 4. NASA EONET ⭐ FREE, NO KEY
Endpoint: https://eonet.gsfc.nasa.gov/api/v3/events
Categories: wildfires, floods, earthquakes, volcanoes, severeStorms, drought, landslides
Example: ?category=wildfires&status=open&limit=10
Fields: title, coordinates, magnitude, date

## 5. GDACS Global Disasters ⭐ FREE, NO KEY
Endpoint: https://www.gdacs.org/gdacsapi/api/events/geteventlist/EVENTS
Covers: Earthquakes, floods, cyclones, droughts globally
Fields: GeoJSON with coordinates, severity, affected population

## 6. UNHCR Population API ⭐ FREE, NO KEY
Turkey: https://api.unhcr.org/population/v1/population/?year=2024&coa=TUR
UK: https://api.unhcr.org/population/v1/population/?year=2024&coa=GBR
Fields: refugees, asylum_seekers, idps, stateless counts

## 7. Overpass API (OpenStreetMap) ⭐ FREE, NO KEY
Endpoint: https://overpass-api.de/api/interpreter
Queries:
- Hospitals: node["amenity"="hospital"](bbox);out body;
- Schools: node["amenity"="school"](bbox);out body;
- Fuel: node["amenity"="fuel"](bbox);out body;
- Shelters: node["emergency"="shelter"](bbox);out body;
- Refugee sites: node["amenity"="refugee_site"](bbox);out body;
Turkey bbox: (36,26,42,45) | UK bbox: (50,-6,56,2)

## 8. UK Police Crime API ⭐ FREE, NO KEY
Endpoint: https://data.police.uk/api/crimes-street/all-crime
Example: ?lat=51.5&lng=-0.1&date=2024-01
Fields: category, latitude, longitude, street, outcome_status

## 9. GDELT News & Events ⭐ FREE, NO KEY (1 req/5sec limit)
Doc API: https://api.gdeltproject.org/api/v2/doc/doc?query=disaster%20turkey&mode=ArtList&format=json
Geo API: https://api.gdeltproject.org/api/v2/geo/geo?query=protest&format=GeoJSON
Fields: title, url, date, tone, coordinates

## 10. IODA Internet Outages ⭐ FREE, NO KEY
Turkey: https://api.ioda.inetintel.cc.gatech.edu/v2/signals/raw/country/TR?from=TIMESTAMP&until=TIMESTAMP
UK: https://api.ioda.inetintel.cc.gatech.edu/v2/signals/raw/country/GB?from=TIMESTAMP&until=TIMESTAMP
Fields: bgp, active_probing, telescope signals

## 11. UK Power Networks ⭐ FREE, NO KEY
Endpoint: https://ukpowernetworks.opendatasoft.com/api/explore/v2.1/catalog/datasets/ukpn-disruption-report/records
Fields: latitude, longitude, postcode, customers_affected, etr

## 12. National Grid UK ⭐ FREE, NO KEY
CSV Feeds for power outages
Fields: latitude, longitude, postcode, incident details

## 13. Open-Meteo Weather ⭐ FREE, NO KEY
Turkey: https://api.open-meteo.com/v1/forecast?latitude=39.9&longitude=32.8&current_weather=true
UK: https://api.open-meteo.com/v1/forecast?latitude=51.5&longitude=-0.1&current_weather=true
Fields: temperature, windspeed, weathercode

## 14. Norway Met Alerts ⭐ FREE, NO KEY
Endpoint: https://api.met.no/weatherapi/metalerts/2.0/all.json
Covers: European marine areas, some land alerts
Fields: polygon coordinates, severity, event type
